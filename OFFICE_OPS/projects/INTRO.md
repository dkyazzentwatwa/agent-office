# Projects Folder Intro

## Purpose
Use this folder to track work that has a defined goal, owner, timeline, or delivery outcome.

## What goes here
- Project briefs
- Status notes
- Milestone tracking
- Active and completed project records

## How to use this folder
Create or update records here when work extends beyond a one-off task. Keep entries clear about owner, scope, status, risks, and next steps.

## Notes for AI agents
Use this folder to understand project context before proposing timelines or assignments. Distinguish between ongoing operations and true project work.
