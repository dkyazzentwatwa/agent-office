# Projects

This folder is optional evidence storage for active work.

## What belongs here

- project briefs
- status notes
- milestone or blocker summaries
- handoff notes for work that spans more than one task

## What does not belong here

- one-off daily triage logs
- speculative planning docs with no active owner
- role prompts or reusable skills

If there are no active projects, this folder can stay empty except for this README.
