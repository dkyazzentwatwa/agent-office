# Quality Folder Intro

## Purpose
Use this folder to define and track what good work looks like across the office.

## What goes here
- Quality standards
- Review criteria
- Performance expectations
- Audit or assurance notes

## How to use this folder
Check this folder when reviewing outputs, creating standards, or tightening consistency. Use it to define quality before judging quality.

## Notes for AI agents
Use this folder to ground reviews in documented expectations. Do not treat personal preference as a quality standard unless the repo says so.
