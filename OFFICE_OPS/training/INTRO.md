# Training Folder Intro

## Purpose
Use this folder to help humans and AI agents learn how the office works and improve over time.

## What goes here
- Onboarding notes
- Playbooks
- Learning guides
- Improvement and enablement materials

## How to use this folder
Start here when joining the workspace, learning a process, or standardizing how a role operates. Keep training docs practical and easy to reuse.

## Notes for AI agents
Use this folder to build context before taking on unfamiliar work. Prefer training guidance when available instead of guessing conventions from scattered files.
