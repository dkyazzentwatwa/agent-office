# Reporting

This folder holds local office reports and daily logs.

## Active folder

- `daily/` - daily triage notes, summaries, escalations, and carry-forward items

## Use this folder when

- creating a daily triage log
- generating a standup or weekly brief from recent activity
- preserving a concise summary for later review

Daily logs may contain private operational data and are gitignored except for `daily/.gitkeep`.
