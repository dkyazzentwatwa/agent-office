# Processes Folder Intro

## Purpose
Use this folder for repeatable operating procedures and standard ways of working.

## What goes here
- SOPs
- Process checklists
- Routine workflows
- Step-by-step operating instructions

## How to use this folder
Start here when you need to run something consistently. Update process docs when the real workflow changes, not just when wording changes.

## Notes for AI agents
Follow documented procedures before improvising. If a process is missing or unclear, note the gap instead of silently inventing a replacement workflow.
