# Coordination Folder Intro

## Purpose
Use this folder to map how work moves between employees, teams, and workflows.

## What goes here
- Handoff maps
- Dependency notes
- Escalation paths
- Cross-functional workflow references

## How to use this folder
Start here when work touches more than one role or when ownership is unclear. Check this folder before assigning multi-step work.

## Notes for AI agents
Use this folder to determine routing order, handoff points, and escalation paths. Prefer existing coordination rules over inventing new workflows.
