# Resources Folder Intro

## Purpose
Use this folder to track capacity, allocation, and the practical limits of the office team.

## What goes here
- Capacity plans
- Utilization records
- Resource allocation notes
- Staffing or bandwidth planning docs

## How to use this folder
Check this folder before assigning major new work or shifting ownership. Use it to spot overload, slack, and staffing gaps.

## Notes for AI agents
Use this folder to balance assignments realistically. Capacity should inform routing, but it should not override role boundaries or required expertise.
